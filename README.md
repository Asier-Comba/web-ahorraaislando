# 🏠 AhorraAislando

Web profesional para empresa de aislamiento térmico y acústico en el País Vasco. La plataforma presenta los servicios de la empresa, explica el proceso de instalación y permite a los clientes contactar fácilmente.

🌐 **[Ver web en producción](https://www.ahorraaislando.es)**

---

## 📋 Servicios que presenta

- **Aislamiento insuflado** — inyectado en cámaras de aire sin obra
- **Corcho proyectado** — material natural para paredes, techos y suelos
- **Tratamientos antihumedad** — soluciones contra condensación y humedad

---

## 🛠️ Stack Tecnológico

| Tecnología | Uso |
|---|---|
| **React 19** | Framework principal |
| **Vite** | Bundler y servidor de desarrollo |
| **Tailwind CSS** | Estilos y diseño responsive |
| **Framer Motion** | Animaciones y transiciones |
| **Radix UI** | Componentes accesibles (modales, tabs, sliders...) |
| **React Router DOM 6** | Navegación entre páginas |
| **React Helmet** | SEO y metadatos dinámicos |

---

## 📄 Páginas

- `/` — Inicio (hero, servicios, beneficios, proceso)
- `/sobre-nosotros` — Sobre la empresa
- `/servicios` — Detalle de servicios
- `/como-funciona` — Proceso paso a paso
- `/contacto` — Formulario de contacto
- Páginas legales: aviso legal, privacidad, cookies

---

## 🚀 Instalación y uso

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Build de producción
npm run build
```
